<p align="center">
T u l i p<br/>
Unlimited, fast, and easy Reverse IP Lookup<br/>
@elliottophellia
</p>

## Requirements
- curl
- jq
- sed
- shuf

## Installation
```
git clone https://github.com/elliottophellia/tulip
cd tulip
chmod +x tulip
sudo mv tulip /usr/local/bin
```

## Usage
```
tulip -d [domain] -v [version] -o [output]
```
